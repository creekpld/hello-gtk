# example GTK app with Rust

## Setup and Run

   ```sh
    # 1.install rust nightly with rustup

    # 2. update 
    rustup update

    # 3. Install the RLS
    rustup component add rls-preview
    rustup component add rust-analysis
    rustup component add rust-src

    # 4. Run
    cd hello-gtk
    cargo run

  ```
